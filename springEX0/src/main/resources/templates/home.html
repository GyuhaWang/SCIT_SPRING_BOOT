Spring 의 주요 개념

  1. DI
    • service class 에서 만든 객체를 @autowired를 사용해 spring 에서 관리하게 함
    • Interface를 사용하면 의존성을 떨어뜨릴 수 있음
      • @Primary,@Service(["name"]) 사용하면 @Autowired @Qualifier("DeluxeTaxiServiceImpl") TransportationService ts; 처럼 사용 가능
  2. IOC
  3. AOP
    • aspect 를 사용한다
    • 매서드를 실행하는 시점에서 aop (proxy)가 가로채서 실행을 합니다.
    • pointcut 부분의 객체를 등록하는 순간에 proxy 객체가 들어감
      • @Before("execution(* net.dsa.web.service.TransportationService.*(..))") public void logBefore(JoinPoint joinpoint) { System.out.println("[콜 수신]"); }
      @After("execution(* net.dsa.web.service.TransportationService.*(..))") public void logAfter(JoinPoint joinpoint) { System.out.println("[결제 처리]"); }
